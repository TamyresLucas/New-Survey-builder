description: Push changes to Azure DevOps repo
---
1. Navigate to the project directory:
   ```bash
   cd "/Users/tamyreslucas/Survey Builder Git/New-Survey-builder-1"
   ```
2. Check the status of the repository:
   ```bash
   git status
   ```
3. Stage all changes:
   ```bash
   git add .
   ```
4. Commit the changes with the provided message:
   ```bash
   git commit -m "<Commit Message>"
   ```
5. Push the changes to the remote repositories:
   ```bash
   git push origin main
   git push github main
   ```
