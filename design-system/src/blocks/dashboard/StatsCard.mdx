# Stats Cards

Statistics cards for displaying metrics and KPIs in dashboards.

## Import

```tsx
import { StatsCard } from "@voxco/design-system/blocks/dashboard"
```

## Voxco Use Cases

| Scenario | Recommended Variant |
|----------|---------------------|
| Results Dashboard | SimpleWithPositiveTrend |
| Abandonment Rate | SimpleWithNegativeTrend |
| Revenue/Billing | MonetaryValue |
| Active Surveys | LargeNumber |
| Status Distribution | WithList |
| Multiple Analytics | WithMultipleMetrics |
| Response Goal | WithProgressBar |

## API

### Props

| Prop | Type | Description |
|------|------|-------------|
| `title` | `string` | Title displayed at the top of the card |
| `value` | `string \| number` | Main highlighted value |
| `trend` | `{ value: number, type: 'positive' \| 'negative' \| 'neutral' }` | Trend indicator |
| `comparison` | `string` | Comparison text (e.g., "vs. last month") |
| `subtitle` | `string` | Subtitle or additional description |
| `items` | `StatsListItem[]` | List of items for list variant |
| `metrics` | `StatsMetric[]` | Metrics grid for multi-metric variant |
| `progress` | `StatsProgressItem[]` | Progress segments for bar variant |
| `className` | `string` | Additional CSS classes |

### Types

```tsx
type TrendType = "positive" | "negative" | "neutral"

interface StatsListItem {
  label: string
  value: string
  amount?: string
  color?: string
}

interface StatsMetric {
  label: string
  value: string
}

interface StatsProgressItem {
  label: string
  percentage: number
  color?: "primary" | "secondary" | "success" | "warning" | "destructive"
}
```
