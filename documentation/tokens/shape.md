# Shape Tokens

## Border Radius

To maintain a consistent and clean aesthetic, a standardized border radius is applied to interactive elements.

-   **Standard Corner Radius**: **4px**
    -   **Tailwind Class**: `rounded` (or custom class resolving to `4px`)
    -   **Usage**: This radius applies specifically to:
        -   **Buttons** (Primary, Secondary, Tertiary, Icon Buttons)
        -   **Input Fields** (Text Fields, Search Bars, Number Inputs)
        -   **Dropdown Fields** (Select menus, Combo boxes)
        -   **Form Elements** (Checkboxes, Radio buttons - if applicable/styled as squares)
