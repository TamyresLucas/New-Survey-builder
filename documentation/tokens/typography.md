# Typography Tokens

## Font Families

The application uses Google Fonts for a clean and modern aesthetic.

-   **Headings & Branding**: **Outfit** (Weights: 300 Light, 500 Medium, 800 Extrabold)
-   **Body & UI Text**: **Open Sans** (Weights: 400 Regular, 500 Medium, 600 SemiBold, 700 Bold, 800 ExtraBold)

## Icons

-   **Library**: **Material Symbols Rounded** (Google Fonts).
-   **Style**: A global style is applied to make all icons **Filled** (`font-variation-settings: 'FILL' 1`) for a consistent and bold appearance.

## Button Typography Standards

To ensure visual consistency, specific typography standards are applied to buttons.

-   **Font Weight**: All buttons with text labels must use **Semibold** weight (600).
    -   **Class**: `font-semibold` (or custom classes mapping to weight 600).
    -   **Scope**: This applies to Primary, Secondary, and Tertiary buttons.
